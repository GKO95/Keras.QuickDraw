# Keras.QuickDraw
Originally a team project of the Department of Electronic Engineering students at Kyonggi University (lead by GKO95), it aims to replicate Google's "Quick, Draw!" online game as close as possible.

## Dependency
* Python: 3.8.7
  * TensorFlow: 2.3.0
    * NumPy: 1.18.5
    * SciPy: 1.4.1
  * PySide2: 5.15.2
  * matplotlib: 3.3.3
    * Pillow: 8.1.0
